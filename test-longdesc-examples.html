❌ Failing Cases

Fails when longdesc is empty

Fails when longdesc points to image

Fails when longdesc is malformed URL

Fails when longdesc is URL but missing filename

Fails with valid-looking but image file extension

✅ Passing Cases

Passes with valid longdesc URL to HTML

Passes when longdesc is not present

Passes with valid external longdesc URL

Passes with URL containing query parameters

Passes with relative URL

Passes with URL containing hash fragment

✅ Hidden Elements (Should Be Skipped)

display:none (inline) should be skipped

visibility:hidden (inline) should be skipped

display:none via class should be skipped

visibility:hidden via class should be skipped

aria-hidden="true" should be skipped
